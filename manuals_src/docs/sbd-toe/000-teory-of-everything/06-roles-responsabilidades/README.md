# Responsabilidades por Capítulo - Visão Geral

Este diretório contém o mapeamento completo de responsabilidades dos 13 roles envolvidos na implementação do **SbD-ToE (Security-by-Design Theory of Everything)**.

## 📊 Estatísticas Globais

- **Total de roles**: 13
- **Total de User Stories mapeadas**: 216 US
- **Capítulos cobertos**: 14 (Cap. 01 a Cap. 14)
- **Cobertura**: Todos os roles têm US ou requisitos associados relevantes
