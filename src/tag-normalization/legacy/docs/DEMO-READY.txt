   source .venv/bin/activate
   python3 demo.py

2. Validar Ficheiros Reais
   
   cd tag_system
   make validate BASE_PATH=../../../manuals_src/docs/sbd-toe LIMIT=5

3. Obter Recomendações
   
   cd tag_system

   cd tag_system

✨ ARQUITETURA IMPLEMENTADA

════════════════════════════════════════════════════════════════════════════════

tag_system/ (modular, SOLID principles)
├── core/canonical_tags.py
│   └─ CanonicalTagsManager
│      ├─ load() - Carrega 489 tags do canonical-tags.yml
│      ├─ get_tag() - Busca tag por nome
│      ├─ find_by_alias() - Encontra tag por alias
│      ├─ get_related_tags() - Retorna tags relacionadas
│      └─ find_similar_tag() - Busca fuzzy match
│
├── validators/validation_engine.py
│   └─ ValidationEngine
│      ├─ validate_file() - Valida tags num ficheiro
│      ├─ validate_batch() - Valida múltiplos ficheiros
│      ├─ extract_frontmatter_tags() - Parse YAML frontmatter
│      └─ Detecta: unknown tags, aliases, case mismatches, duplicates
│
├── recommenders/recommendation_engine.py
│   └─ RecommendationEngine
│      ├─ recommend_tags() - Recomenda tags baseado em conteúdo
│      ├─ extract_content_text() - Extrai keywords
│      ├─ get_parent_context_tags() - Context aware
│      └─ Scoring: keyword(30%) + semantic(25%) + context(20%)
│
└── cli/main_cli.py
    └─ TagSystemCLI
       ├─ validate - Valida tags existentes
       ├─ recommend - Recomenda tags novas
       └─ audit - Validação + Recomendações
