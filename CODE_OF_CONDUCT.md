# Código de Conduta – Security by Design – Theory of Everything

Este projeto adota o [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Compromisso

Promover um ambiente inclusivo, colaborativo e respeitador em todas as interações.  
Não são toleradas condutas discriminatórias, assédio, insultos ou intimidação.

## Aplicabilidade

Aplica-se a todas as plataformas associadas ao projeto:
- Repositórios GitHub e discussões
- Issues, Pull Requests e Revisões
- Fóruns, eventos e comunicações oficiais

## Denúncias

Comportamentos inadequados podem ser reportados para:

📧 **security@shiftleft.pt**

Todas as denúncias são tratadas de forma confidencial.  
A manutenção deste código é condição de participação no projeto.
