SETUP (ONE-TIME)

Step 1: Install Ollama and Mistral
  $ brew install ollama
  $ ollama pull mistral

Step 2: Setup Python environment
  $ cd src/manual-rag
  $ bash setup.sh

Step 3: Build the index (~2 minutes)
  $ source rag_env/bin/activate
  $ python3 build_index.py

USAGE (EVERY SESSION)

Always start in a new terminal:

Terminal 1: Start Ollama (runs continuously)
  $ ollama serve

Terminal 2: Use the RAG system
  $ cd src/manual-rag
  $ source rag_env/bin/activate

Then run commands:

  # Search for related documents
  $ python3 __main__.py search "secure coding practices"

  # Get tag suggestions for a file
  $ python3 __main__.py tag "010-sbd-manual/06-desenvolvimento-seguro/01-intro.md"

  # Analyze gaps in a document
  $ python3 __main__.py gaps "010-sbd-manual/06-desenvolvimento-seguro/01-intro.md"

  # Find cross-references to a topic
  $ python3 __main__.py xref "authentication"

  # Check system health
  $ python3 __main__.py health

  # Get help
  $ python3 __main__.py --help

TROUBLESHOOTING

  "Cannot connect to Ollama"
    → Is ollama serve running in another terminal?
    → Check: curl http://localhost:11434/api/tags
    → Fix: brew install ollama && ollama pull mistral

  "Index not found"
    → Run: python3 build_index.py
    → Wait for ~2 minutes to complete

  "ModuleNotFoundError"
    → Activate venv: source rag_env/bin/activate
    → Reinstall: pip install -r requirements.txt
