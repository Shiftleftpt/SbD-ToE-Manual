# Capítulos Basilares do Manual SbD – Teoria de Tudo {capitulos-basilares}

Nem todos os capítulos do manual têm o mesmo peso estrutural.

Alguns são considerados **basilares** — isto é, representam os **fundamentos essenciais** para que as restantes práticas de segurança possam ser:

- Proporcionais ao risco
- Auditáveis
- Técnicas e organizacionalmente coerentes

> Sem estas fundações, a aplicação de Security by Design pode tornar-se ineficaz ou meramente superficial.

## 📌 Como usar esta informação {capitulos-basilares#como_usar_esta_informacao}

- Estes capítulos devem ser **prioritários** na leitura, formação e auditoria.
- A sua ausência ou má implementação **compromete a eficácia geral do modelo**.
- São também a base para **modelos de maturidade** como SAMM, SSDF e BSIMM.

---

## 🔗 Sugestão {capitulos-basilares#sugestao}

Rever estes capítulos antes de aplicar qualquer prática isolada de segurança.  
Eles são o **ponto de partida lógico** para qualquer programa estruturado de Security by Design.
