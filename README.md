# Security by Design – Theory of Everything

Este é um manual prático e prescritivo de Security by Design, orientado ao ciclo de vida do software.

## Objetivo
Promover a adoção de práticas seguras desde o design até à entrega, com exemplos hands-on, orientações por papel, e integração com frameworks reconhecidas (OWASP, NIST, SLSA, etc.).

## Idiomas
- 🇵🇹 Português (default)
- 🇬🇧 English (em preparação)

## Estrutura
Organizado por 14 áreas principais de segurança aplicacional. Cada uma contém:
- O que é
- Como fazer
- Papéis responsáveis
- Ferramentas e exemplos
- Checklists e templates

## Deploy
Este site é gerado com [Docusaurus](https://docusaurus.io) e publicado via GitHub Pages.
