🎯 COMANDOS PRINCIPAIS

# Demo Interativa
$ make demo

# Gerar Relatório (scanner)
$ make scan
$ make scan BASE_PATH=/custom/path REPORT=my-report.json

# Visualizar Relatório
$ make view-report              ← Resumo completo
$ make view-summary             ← Apenas resumo
$ make view-no-tags             ← Ficheiros sem tags
$ make export-csv              ← Exportar para CSV

# CLI Direto
$ make validate BASE_PATH=...
$ make recommend BASE_PATH=...
$ make audit BASE_PATH=...
