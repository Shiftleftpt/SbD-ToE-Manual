# Pull Request Templates

Esta pasta ativa o seletor de múltiplos templates de PR no GitHub.

- `conteudo-capitulo.md` — novo conteúdo/melhoria de capítulo
- `revisao-editorial.md` — revisão editorial/estilo
- `correcao-bug.md` — correções de build/MDX/site
- `atualizacao-checklists.md` — alterações a `canon/20-*`
- `infraestrutura-build.md` — alterações a CI/build/config

> Dica: podes pré-selecionar um template via URL:
> `.../compare/master...branch?template=revisao-editorial.md`
